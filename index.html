box-sizing: content-box

sizing = content

box-sizing: border-box

sizing = content+padding+border
This is the browser default. The width and height we set for the box refer (as the name suggests) only to the content
The width and height we set for the box refer to everything: content+border+padding

box-sizing:content-box;

height:330px;
padding:50px

}		 #boxModel {
box-sizing:border-box;
width:330px;
height:330px;
padding:50px

}
